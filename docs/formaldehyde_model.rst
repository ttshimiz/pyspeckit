Formaldehyde Models
===================

The Formaldehyde model is based on the :doc:`hyperfine_model`.  There is also a
mm-line fitting module that uses preconstructed model grids to fit temperature
from multiple transitions.

There is a formaldehyde fitter wrapper; see :doc:`wrappers`.
